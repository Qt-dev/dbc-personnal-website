For

The for loop has different implementation in Ruby according to the situation (each, until, etc).

This is the go-to tool in Javascript. i is the index of the loop that you're in.

for (var i=0; i<maximum; i++){
  //Your code here.
};

For..in

For..in

This iterates through a fixed number of objects.
Here, i is the index of the loop and we have it set to iterate through 0-5.

for i in 0..5
  #Your code here.
end

This iterates over the property of an object without taking the order into account.
Here, i is the identification of the property (name). If the object is an array, it will instead be the index.

for (var i in object){
  if (object.hasOwnProperty(i)){
    //Your code here
  }
}
